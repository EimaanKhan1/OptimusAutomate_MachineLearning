# Machine Learning Internship Tasks

This project implements all four tasks from the internship brief using reproducible, offline-friendly datasets and end-to-end machine learning pipelines.

## Tasks Included

1. Titanic Survival Prediction
2. House Price Prediction
3. Customer Churn Prediction
4. Sentiment Analysis on Product Reviews

## How to Run

```bash
python run_all_tasks.py
```

To write outputs to a custom folder:

```bash
python run_all_tasks.py --output-dir outputs
```

## Output

Each task prints a metric summary to the console and saves visualizations into the output directory.
